Hello, welcome to a BlockSite!

BlockSites are "web"-sites that live entirely on Sui

No web involved! The content of this blocksite is loaded from an object on Sui. The object contains all the resources necessary to render this page. To browse a blocksite, you just need its Sui Object ID. No wallet is needed to browse blocksites, and publishing to Sui is extremely inexpensive!

How does it work?

  • Each blocksite is represented by objects on Sui, which contain a version of the blocksite
  • Once the blocksite portal is loaded, it downloads a service worker that gets the contents of the objects from the Sui blockchain and serves them to the browser.
  • People can browse any blocksite by just typing its object ID (in base36 encoding) as a subdomain to the blocksite portal url.
  • The service worker in the portal then fetches the objects from the full node and displays them. That's it!

How can i publish a blocksite?

By using the site builder tool.
