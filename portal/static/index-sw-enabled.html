Blocksites!

What is a Blocksite?

BlockSites are "web"-sites that live entirely on Sui, no web involved! The content of a blocksite is loaded from an object on Sui, as the object contains all the resources necessary to render the site. To browse a blocksite, you just need its Sui Object ID or SuiNS name. No wallet is needed to browse blocksites, they are accessible to everyone!
