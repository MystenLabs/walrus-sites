Hello, welcome to a BlockSite!

BlockSites are "web"-sites that live entirely on Sui

No web involved! The content of this blocksite is loaded from an object on Sui. The object contains all the resources necessary to render this page. To browse a blocksite, you just need its Sui Object ID. No wallet is needed to browse blocksites, and publishing to Sui is extremely inexpensive!

How does it work?

  • Each blocksite is represented by an object on Sui, which contains a compressed version of the blocksite (in this case, the html and css).
  • Once the blocksite portal is loaded, it downloads some javascript code to get the contents of the object from the Sui blockchain, decompress it and render them in an iframe.
  • People can browse any blocksite by just typing its object ID after a # in the blocksite portal url.
  • The javascript in the portal then fetches the object from the full node, and displays it. That's it!

How can i publish a blocksite?

Also easy: Just load up the blocksite portal, connect your wallet, and drag-and-drop the inlined html/css/js. Accept the transaction cost, and you have a blocksite forever! (For more complex websites, a bit more tooling is needed). In case you are unhappy with the blocksite, you can always update it!
